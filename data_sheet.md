# Datasheet

## Motivation

- Imperial Executive Education created 8 datasets which were inputs and outputs of 8 synethic black box functions for this Capstone Project.
- The Capstone project was an opportunity to primarily apply Bayesian Optinmisation Gaussian Processes techniques to a maximisation problems,
- developing strategies week to week to understand each black box function better and identify it's global maximum. 
 
## Composition

- An initial dataset was made available for each function. Each function's input data was represented by 10-40 known (x,y) data points. The x data points
- ranged from 2D to 8D arrays with corresponding y values which were all 1D.
- For example, function 1's initially provided input data comprised of 20 x values represented in a 2D array with 20 1D output y values
- whereas functiom 8's initially provided input data comprised of 40 x values represented in a 8D array with 40 1D  values.
- The input data points (x) were all bound in the range of 0.0 to 1.0 to a maximum of 6 decimal points and the output data points (y) were all float data points
- representing the each function's output value for a given x input. Technically there was no bound to the y value which could take any value -/+ infinity
- There was also no missing data in the data provided.
- The datasets provided for each function also did not hold any confidential data as these were observations created for this capstons project's 8 synthetic functions.

## Collection process, Preprocessing/cleaning/labelling

- The initial datasets were prepared by the Imperial Exective Education team and provided to all students on this cohort so we all had the same starting points.  

## Uses

  The initial daatsets were specifically provided for this capstone project for this student cohort. They can be used again for other student cohorts and used
  in other machine learning courses.
